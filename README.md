## インストール方法

以下の手順でプロジェクトをローカル環境にインストールしてください。

```
ディレクトリを作成し移動
mkdir repository
cd repository

リポジトリをクローン
git clone https://github.com/j329nish/Authenticity-check.git

ディレクトリに移動
cd repository

依存関係をインストール
pip install -r requirements.txt
```
